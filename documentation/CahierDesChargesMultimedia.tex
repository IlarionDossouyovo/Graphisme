\chapter*{Préambule}
\addcontentsline{toc}{chapter}{Préambule}
Ce cahier des charges définit les spécifications pour la production de tous les médias visuels et audiovisuels de E-Graphisme by ELECTRON. Il sert de référence pour la création des images, vidéos, animations, voix off et effets sonores.

\bigskip
\textbf{Classification :} Interne - Partageable\\
\textbf{Version :} 1.0\\
\textbf{Date :} Juillet 2026

\chapter{OBJECTIF}

\section{Objectif Global}
Créer une identité visuelle premium, homogène et immersive couvrant l'ensemble de la plateforme, des supports commerciaux et des contenus marketing.

\item Application Web
\item ERP, CRM

\chapter{PRÉCONISATIONS TECHNIQUES}

\section{Images}
\begin{itemize}
\item Format : PNG, SVG, JPG, WebP
\item Résolution : 72 DPI (web), 300 DPI (impression)
\item Couleurs : sRGB, CMJN pour impression
\item Optimisation : Compression sans perte
\end{itemize}

\section{Vidéos}
\begin{itemize}
\item Codec : H.264, H.265
\item Débit : 8-15 Mbps (1080p), 20-35 Mbps (4K)
\item Audio : AAC 256kbps
\item Images/seconde : 24, 30, 60 FPS
\end{itemize}

\section{Animations}
\begin{itemize}
\item Format : Lottie, GIF, MP4, WebM
\item Cadence : 60 FPS
\item Durée : 2-10 secondes (loops)
\item Optimisation : < 2MB pour web
\end{itemize}
