\chapter{AI CREATIVE STUDIO}

\section{Présentation}
Le AI Creative Studio est le cœur créatif de la plateforme. Il permet à toute personne, même sans compétences en graphisme, de produire des œuvres originales destinées à la decoration ou à la vente.

Le studio fonctionne entièrement avec les modèles IA locaux exécutés via Ollama, garantissant confidentialité, contrôle et indépendance.

\section{Objectifs}
Le Studio permet de :
\begin{itemize}
\item créer des tableaux decoratifs ;
\item produire des œuvres numériques ;
\item personnalisés les créations ;
\item générer des affiches ;
\item préparer les fichiers pour l'impression ;
\item publier les œuvres sur la boutique intégrée.
\end{itemize}

\chapter{AI INTERIOR DESIGNER}

\section{Présentation}
Le client peut importer une photo de son intérieur.

\section{Analyse de la Pièce}
L'IA analyse :
\begin{itemize}
\item les dimensions de la pièce ;
\item les couleurs dominantes ;
\item le style du mobilier ;
\item la luminosité ;
\item l'emplacement des murs disponibles.
\end{itemize}

\section{Proposition d'Œuvres}
Elle propose ensuite plusieurs œuvres adaptées et crée une simulation réaliste de leur intégration dans la pièce.
